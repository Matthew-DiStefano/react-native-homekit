# react-native-homekit

## Installation

```sh
npm install react-native-homekit
```

## Usage

```js
import Homekit from "react-native-homekit";

// ...

const result = await Homekit.addHome("Nazir's Home");

const result = await Homekit.removeHome("Nazir's Home");
```
